1.2 - Tipagem dinâmica

>>> a = 1
>>> type(a)
<class 'int'>
>>> a = 'abacaxi'
>>> type(a)
<class 'str'>
>>> a = 1.0
>>> type(a)
<class 'float'>

1.2.1 - Tipagem dinâmica

>>> a = [1, 2.2, '23', 'ola', None]
>>> type(a)
<class 'list'>
>>> a = {1,2}
>>> type(a)
<class 'set'>
>>> a = {1:2}
>>> type(a)
<class 'dict'>

1.5.1 - É legal mexer com listas

liguagem = "python"
caracter = list(liguagem)
print(caracter)

palavra = "".join(caracter)
print(palavra)

1.5.2 - Lista é mutável

matrix = matrix*3
print(matrix)
matrix[1] = 2
print(matrix)
